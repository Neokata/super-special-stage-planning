# 03 — Friday kickoff event

**Scope: this file covers the Friday night kickoff event only
(6:00–9:00 PM CT).** It does NOT cover Friday or Saturday driving
operations. See `04-operations.md` for the full weekend.

## Schedule

| Time (CT) | Activity |
|---|---|
| 5:00 PM | Staff arrives, lot setup, vendor load-in |
| 5:30 PM | Gates open for in-park parking ($50) and reserved lot ($30) |
| 6:00 PM | Kickoff event officially begins |
| 6:15 PM | DJ starts (live set, no battle format) |
| 6:30 PM | Breakdancer showcase / cypher |
| 7:30 PM | Raffle drawing (round 1) |
| 8:30 PM | Raffle drawing (round 2 / grand prizes) |
| 9:00 PM | Event ends, vendors out by 9:30 PM |
